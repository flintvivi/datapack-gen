Datapacks 101

Minecraft datapacks add or override data-driven game behavior without mods.

  • Each pack needs a pack.mcmeta file.
  • Content lives under data/<namespace>/.
  • Functions are plain .mcfunction command files.
Learn more on Minecraft Wiki
